# ⚙️ Smart Telecom Complaints - Backend API & AI Engine
This repository contains the backend infrastructure for the Smart Telecom Complaints System.

### 🛠️ Technologies Used:
- **Core APIs:** PHP 8.3 (RESTful architecture)
- **Database:** MySQL 8.0 (Relational schema with triggers)
- **AI Classification:** Python, Scikit-Learn, Google Gemini API
- **Notifications:** Firebase Cloud Messaging (FCM v1)
- **Deployment:** Google Cloud Platform (Ubuntu 24.04 LTS)

*Note: Security keys, database credentials, and user uploads are ignored in this repository for security purposes.*
